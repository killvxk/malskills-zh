# x86-64 Reference

## Register Table

| Register | 32-bit | 16-bit | 8-bit | Role |
|---|---|---|---|---|
| `rax` | `eax` | `ax` | `al` | Return value, accumulator |
| `rbx` | `ebx` | `bx` | `bl` | Callee-saved general purpose |
| `rcx` | `ecx` | `cx` | `cl` | Arg 4, loop counter, shift count |
| `rdx` | `edx` | `dx` | `dl` | Arg 3, high half of mul/div |
| `rsi` | `esi` | `si` | `sil` | Arg 2, string source |
| `rdi` | `edi` | `di` | `dil` | Arg 1, string destination |
| `rbp` | `ebp` | `bp` | `bpl` | Frame pointer (callee-saved) |
| `rsp` | `esp` | `sp` | `spl` | Stack pointer |
| `r8`–`r11` | `r8d`–`r11d` | `r8w`–`r11w` | `r8b`–`r11b` | Args 5–6 (`r8`/`r9`), caller-saved |
| `r12`–`r15` | `r12d`–`r15d` | `r12w`–`r15w` | `r12b`–`r15b` | Callee-saved |

**SIMD**: `xmm0`–`xmm7` caller-saved (args/return); `xmm8`–`xmm15` callee-saved on Windows, caller-saved on Linux.

## Instruction Selection Guide

### Arithmetic

| Goal | Preferred | Avoid |
|---|---|---|
| `x * 2` | `lea rax, [rdi*2]` or `shl rdi, 1` | `imul rdi, 2` |
| `x * 3` | `lea rax, [rdi + rdi*2]` | `imul rdi, 3` |
| `x * 9` | `lea rax, [rdi + rdi*8]` | `imul rdi, 9` |
| `x / 2` (signed) | `sar rdi, 1` | `idiv` |
| `x / 2` (unsigned) | `shr rdi, 1` | `div` |
| `x % power-of-2` | `and rdi, (N-1)` | `div` |
| `abs(x)` | `mov rax, rdi` / `neg rax` / `cmovl rax, rdi` | branch |

### Conditional / Branchless

```nasm
; if (a < b) result = a else result = b  →  min(a, b)
cmp     rdi, rsi
cmovg   rdi, rsi        ; rdi = min(rdi, rsi)
mov     rax, rdi
ret

; bool x = (a == b)
xor     eax, eax        ; zero-extend to rax — faster than xor rax,rax
cmp     rdi, rsi
sete    al
ret
```

### Memory Access and Alignment

- **16-byte alignment** required for `movaps`, `movdqa` — use `movups`/`movdqu` for unaligned
- **32-byte alignment** required for `vmovaps` / `vmovdqa` (AVX)
- **64-byte alignment** for AVX-512

```nasm
; Align a label/data to 16 bytes
align 16
.hot_loop:
    ; loop body
```

### Zero / Sign Extension

```nasm
movzx   eax, byte [rdi]     ; zero-extend byte → 32-bit (also clears upper 32)
movzx   eax, word [rdi]     ; zero-extend word → 32-bit
movsx   rax, dword [rdi]    ; sign-extend 32-bit → 64-bit
cdqe                        ; sign-extend eax → rax
```

## SSE2 / AVX2 Patterns

### Horizontal Sum (4 floats → 1)

```nasm
; xmm0 = [a, b, c, d] → xmm0[0] = a+b+c+d
haddps  xmm0, xmm0          ; [a+b, c+d, a+b, c+d]
haddps  xmm0, xmm0          ; [a+b+c+d, ...]
```

### Byte-Level Search (find zero byte)

```nasm
; Check 16 bytes for zero byte using PCMPEQB + PMOVMSKB
movdqu  xmm0, [rdi]
pxor    xmm1, xmm1
pcmpeqb xmm0, xmm1          ; 0xFF per byte == 0
pmovmskb eax, xmm0           ; bitmask: bit N set if byte N was 0
test    eax, eax
jnz     .found_zero
```

### AVX2 — 32-byte integer compare

```nasm
vmovdqu  ymm0, [rdi]
vpcmpeqb ymm0, ymm0, ymm1   ; ymm1 = value to find
vpmovmskb eax, ymm0
```

## ABI Edge Cases

- **varargs**: `al` must hold the number of used XMM registers before calling a variadic function
- **`syscall` clobbers**: `rcx`, `r11` (used internally); use `r10` instead of `rcx` for arg 4
- **Red zone**: 128 bytes below `rsp` are safe in leaf functions (signal-safe only in user space)
- **TLS** (`fs` / `gs` segment): do not modify segment registers
- **Return address**: always `[rsp]` at function entry; push/pop disturbs this
